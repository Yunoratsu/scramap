# Google Maps Business Search Automation

A Python tool that automates Google Maps searches in the background by combining business names with custom locations or regions. It is useful for lead generation, market research, business prospecting, and workflow automation.

## Features

* Background Google Maps search automation
* Search using business names and custom locations
* Supports batch processing of multiple search terms
* Fast and lightweight
* Easy to customize
* Built with Python

## How It Works

The application generates search queries by combining business names with locations (for example, "Dentist New York" or "Restaurant Los Angeles") and automatically opens Google Maps in the background to perform the searches.

This project is designed to simplify repetitive search tasks and can be integrated into larger data collection or automation workflows.

## Requirements

* Python 3.10+
* Google Chrome (or another supported browser)
* Internet connection

## Installation

```bash
git clone https://github.com/Yunoratsu/scramap.git

cd scramap

pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python main.py
```

Configure your list of business names and locations according to your project structure.
